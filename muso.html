Muso

Welcome to Muso!

This is a game modelled on the excellent app DuoLingo but designed for learning music theory. You wont find information on music theory here, the application is for practice to make reading music more natural.

To use it simply try the exercises, if you do well then try the challenge mode of each exercise where you will be asked more questions that get harder towards the end. The recap tests randomly pick questions from the exercises to keep you on your toes. For example Test 1-4 could ask any question from the first 4 exercises. Remember - practice often!

Apologies to phone users, I intend to make a non-flash version that will run in any browser in future.
